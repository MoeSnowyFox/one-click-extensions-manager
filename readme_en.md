# Auto Extension Manager

English | [中文](./readme.md)

![Options](pic-options.png)

An intelligent Chrome extension to automate the management of your Chrome extensions.

This project is a fork of [one-click-extensions-manager](https://github.com/hankxdev/one-click-extensions-manager)
with enhanced extension management capabilities, allowing you to automatically switch extensions based on the website URL.

This extension is currently developed and maintained by [MoeSnowyFox](https://github.com/MoeSnowyFox) and [copilot](https://github.com/features/copilot).

## Installation

You can compile and run this extension manually. It has not been uploaded to the Extension Store yet.
